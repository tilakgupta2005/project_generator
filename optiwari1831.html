Project Idea: "C-Powered Study Schedule & Progress Tracker"

Helps students effectively manage their study time, visualize progress, and identify free slots for new learning by automating schedule tracking and reporting.

Tech Stack

HTML CSS C Excel

Short Guide to Get Started

  1. **Design Your Study Data in Excel:** Create an Excel spreadsheet to log study subjects, estimated time per task, actual time spent, due dates, and priority levels.
  2. **Develop a C Program for Schedule Logic:** Write a C program to read your study data (e.g., exported as a CSV from Excel). Implement functions to calculate progress, highlight urgent tasks, and identify free time based on your entries.
  3. **Generate an HTML Study Report:** Modify your C program to dynamically output the processed study schedule and progress information into a well-structured static HTML file.
  4. **Style the Report with CSS:** Apply CSS to your generated HTML file to create a clean, readable, and visually organized display of your schedule, progress bars, and task priorities.
  5. **Automate & Integrate:** Create a simple workflow where you update your Excel sheet, run your C program to regenerate the HTML report, and then open the HTML file in a browser to review your up-to-date study plan.
