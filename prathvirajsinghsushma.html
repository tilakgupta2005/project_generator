Project Idea: "RGVPSyllabusOptimizer"

Helps students balance RGPV syllabus requirements (assignments, MSTs, GPA) with personal upskilling goals, reducing stress and improving time management for better sleep.

Tech Stack

Python Figma Excel

Short Guide to Get Started

  1. 1. Map RGPV Syllabus: Use Python to create a script that lists all RGPV subjects, their assignments, MSTs, and weightages. Store this data, perhaps initially in a structured Excel file.
  2. 2. Define Upskilling Goals: Identify specific skills you want to learn (e.g., Python libraries, Figma design principles). Quantify them into 'learning units' (e.g., hours of practice, tutorials completed).
  3. 3. Design the Interface (Figma): Create a user-friendly interface design in Figma. This will visualize how you input your RGPV tasks and upskilling goals, and how the optimized schedule is presented.
  4. 4. Develop the Optimizer (Python): Write a Python script that takes your RGPV task deadlines and your upskilling goals as input. The script will then suggest a daily/weekly schedule, prioritizing RGPV tasks while allocating dedicated time for upskilling, ensuring breaks and sleep.
  5. 5. Visualize & Refine: Use Excel to track your progress against the generated schedule. Continuously feed this data back into the Python script to refine the optimization logic based on your actual progress and energy levels.
