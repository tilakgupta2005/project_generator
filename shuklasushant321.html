Project Idea: "Academic & Tech Learning Tracker"

Helps balance college coursework and self-learning of new technologies by providing a structured way to track progress, deadlines, and resources.

Tech Stack

Excel C++ front end web development

Short Guide to Get Started

  1. Design a spreadsheet in Excel to list academic subjects, assignments, deadlines, and separate sections for tech learning goals, resources, and progress.
  2. Create a basic front-end web application (using HTML, CSS, and JavaScript) to display this data in a more user-friendly, visual format.
  3. Develop a C++ program to potentially read/write data to/from the Excel file (e.g., using file I/O to parse CSV exports of your Excel sheet) or to perform data analysis on tracked progress.
  4. Integrate the C++ program's output (e.g., summarized progress reports, upcoming deadlines) into the front-end web application, potentially via simple text file exchange or by having the web app fetch pre-generated reports.
  5. Refine the web interface for clear visualization of academic load versus tech learning activities, allowing easy updates and tracking.
