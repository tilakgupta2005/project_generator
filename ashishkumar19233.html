Project Idea: "Effort Tracker CLI"

Helps users meticulously track their dedicated work hours on specific tasks or projects, promoting a culture of hard work and accountability.

Tech Stack

C++ javascript

Short Guide to Get Started

  1. Develop a command-line interface (CLI) application in C++ to record work sessions.
  2. Implement features to start, stop, and pause tracking for different tasks (e.g., 'coding', 'debugging', 'learning').
  3. Store session data (task name, start time, end time, duration) in a simple text file or CSV.
  4. Create a separate web interface using JavaScript (e.g., with a simple local server or static HTML/JS) to visualize the tracked data.
  5. The web interface should display daily/weekly summaries and allow users to export their hard work data.
