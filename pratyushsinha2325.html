Project Idea: "Daily Time Tracker & Analyzer"

Helps you understand how you spend your time, identify time-wasting activities (like reels!), and allocate more time to skill development and personal goals.

Tech Stack

C Excel

Short Guide to Get Started

  1. Step 1: Design a simple data structure in C to record activities. Include fields like 'Activity Name', 'Start Time', 'End Time', and 'Category' (e.g., 'Study', 'Reels', 'Personal').
  2. Step 2: Write C code to prompt the user to enter their activities throughout the day. Store this data in a C array or linked list.
  3. Step 3: Implement a function in C to calculate the duration of each activity (End Time - Start Time).
  4. Step 4: Write C code to export the activity data (Activity Name, Start Time, End Time, Duration, Category) to a CSV (Comma Separated Values) file. This is a text file that Excel can easily open.
  5. Step 5: Open the CSV file in Excel. Use Excel's charting and filtering features to visualize your time allocation by category. Create a pie chart to show percentages, and filter to see how much time you spend on 'Reels' specifically.
