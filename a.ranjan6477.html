Project Idea: "Study Scheduler"

Helps students organize their study time effectively, preventing procrastination and improving time management.

Tech Stack

C++ Excel c Basic Java script

Short Guide to Get Started

  1. Use C++ or C to create a console application that takes study tasks as input (subject, time needed, deadline).
  2. Implement features for adding, deleting, and listing tasks. Store the tasks in a simple text file or CSV format.
  3. Use Excel to create a visual representation of the schedule. Import the task data from the text/CSV file.
  4. Use basic JavaScript within Excel (through VBA or a similar approach) to automatically highlight overdue tasks or calculate remaining study hours.
  5. Enhance the C++ application to generate the CSV/text file in a format easily importable into Excel.
