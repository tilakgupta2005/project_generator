Project Idea: "C Console Goal Tracker"

This project helps you track your daily or weekly goals directly from the command line, providing a simple way to visualize your progress and encourage consistency by making your goals visible and manageable.

Tech Stack

C

Short Guide to Get Started

  1. Design a `struct` in C to represent a goal, including fields like `name` (string), `status` (e.g., 'pending', 'completed'), and maybe a `due_date` (string or simple int for day count).
  2. Implement functions for basic operations: `add_goal(const char* name)` to create a new goal, `list_goals()` to display all goals, and `mark_goal_complete(int id)` to update a goal's status.
  3. Utilize file I/O in C (e.g., `fopen`, `fprintf`, `fscanf`) to save and load your goals from a simple text file, ensuring your data persists even after the program closes.
  4. Create a simple command-line interface (CLI) using `printf` and `scanf` to allow users to interact with the tracker (e.g., '1: Add Goal', '2: List Goals', '3: Mark Complete', '4: Exit').
  5. Implement a main loop that displays the menu, takes user input, and calls the appropriate functions, making sure to handle common input errors gracefully.
