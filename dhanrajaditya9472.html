Project Idea: "Personal Time Manager & Report Generator"

Helps individuals track their time spent on various activities and generates simple reports to understand time allocation, addressing the challenge of finding time for learning by highlighting existing time usage.

Tech Stack

C++ Excel HTML CSS

Short Guide to Get Started

  1. Develop a C++ console application to log time entries. Each entry should include activity name, start time, and end time.
  2. Implement C++ functions to calculate the duration of each activity and total time spent per activity.
  3. Create C++ code to export the logged time data into a CSV format, which Excel can easily read.
  4. Design a simple HTML page with CSS for a visually appealing presentation of the project's purpose and how to use it.
  5. Manually import the generated CSV into Excel to create basic charts or summary tables of time usage, then use the HTML page as an introductory guide.
